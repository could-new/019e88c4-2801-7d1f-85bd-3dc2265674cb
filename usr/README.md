# Organization Payment Tracker

A cross-platform Flutter application for tracking payments, expenses, and financial transactions within an organization. 

## Features

* **Dashboard**: View an overview of total payments, pending amounts, and recent transactions.
* **Payment Logging**: Add new payments with details such as amount, date, payee/payer, and category.
* **Transaction History**: Scrollable list of all historical payments.
* **Cross-platform**: Works on Web, Desktop, and Mobile.

## Getting Started

1. Ensure you have the Flutter SDK installed.
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the app on your preferred device.

## Tech Stack

* **Framework**: Flutter
* **Language**: Dart
